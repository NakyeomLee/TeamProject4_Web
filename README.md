<h1 align="center">
  WEB프로젝트 (NO MORE SHINSA)
</h1>
<p align="center">의류 쇼핑몰 사이트</p>
<p align="center">
  기존 쇼핑몰의 필터 기능을 더 쉽고 직관적이게 이용할 수 있도록 개선한 web 홈페이지 <br>
  간단한 옵션 선택 구성의 설문 기능을 통해 맞춤 의류 추천 기능을 제공하여 <br>
  사용자가 구매 고민을 덜고 최적의 의류를 찾아서 구매할 수 있음
</p>

---
